## [1.0.0] - 2026-03-14

### 🎉 GemDeck v1.0.0 "Cellsica Release"

GemDeckは、あなた専用の最高なAIスペシャリスト（GemやClaudeなど）を管理するための、カスタマイズ可能なパーソナルAIランチャーです。
「Cellsica」ブランドとして最初の公式リリースを迎えました！

### ✨ 主な機能と特徴

- **AIランチャー機能**
  - AIサービスへのリンクを直感的な「セキュリティカード」風デザインのカードとして管理。
  - カードクリック時のアイコン部分で、すぐに別タブでAIを起動可能。
  - カスタムアイコンの設定対応、及びGoogle Drive画像のURL自動変換。

- **モバイルファースト・PWA対応**
  - PWA（Progressive Web App）としてインストール可能。スマホのホーム画面に専用アプリアイコンを配置。
  - Web Share Target APIを用いた、ブラウザからのURL直接共有によるGem追加機能。（※Android OS・ブラウザ仕様に準拠）
  - Tailwind CSSによるダークモード対応と完全なレスポンシブデザイン。

- **ユーザーデータの管理**
  - **インポート & エクスポート**: JSONファイルで構成データを手軽にバックアップ・復元。
  - **ピン留め (Pin)**: 頻繁に使うAIカードを一番上に固定。
  - **ファクトリーリセット**: 全てのローカルデータを消去する機能。
  - データは全てブラウザの localStorage 等に保存されるため、安全かつ高速に動作。

- **UX/UIデザインの追求**
  - スムーズなアニメーション、ホバー時の一工夫を取り入れたインタラクティブなUI。
  - はじめて使うユーザーにも優しい「Welcome（チュートリアル）」画面の実装。

### 🚀 デプロイとインフラ

- GitHub Actionsを利用したCI/CDパイプライン。
- GitHub Pagesを介した高速な静的配信。
